# DevTrack AI

DevTrack AI is a full-stack web application that analyzes GitHub profiles and provides actionable insights to improve a developer’s portfolio.



## Features

* Analyze any public GitHub profile
* Repository statistics (repos, stars, languages)
* Smart suggestions based on activity
* GitHub OAuth login integration
* Real-time API responses
* Modern UI with smooth UX



## Tech Stack

**Frontend:**

* HTML, CSS, JavaScript

**Backend:**

* FastAPI (Python)

**APIs & Services:**

* GitHub REST API
* OAuth Authentication

**Deployment:**

* Render (Frontend + Backend)



## Live Demo

* Frontend: https://devtrack-frontend-lkmi.onrender.com
* Backend API: https://devtrack-backend-z7lc.onrender.com



## How It Works

1. User enters a GitHub username
2. Backend fetches repository data via GitHub API
3. Data is analyzed (repos, stars, languages)
4. Suggestions are generated
5. Results are displayed in a clean UI



## Authentication

Users can log in via GitHub OAuth to fetch their profile securely.
